## 📱 Smartphone Analytics Dashboard

The **Smartphone Analytics Dashboard** is a Power BI project that analyzes smartphone sales data and provides key business insights through interactive visualizations.

It helps track **total sales, average price, brand-wise performance, and customer ratings**, enabling better understanding of market trends and business performance.

The dashboard includes **KPIs, charts, and filters** to compare brands like Apple, Samsung, and Xiaomi and explore data dynamically.

### 🔹 Key Features

* Total Sales Analysis
* Average Price Tracking
* Brand-wise Sales Comparison
* Customer Rating Analysis
* Interactive Filters & Visuals

### 🛠️ Tools Used

Power BI | DAX | Data Modeling | Data Visualization 🚀📊
